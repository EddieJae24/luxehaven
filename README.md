## Description

💋 About LuxeHaven

LuxeHaven is a modern lifestyle platform dedicated to helping clients experience luxury in every detail — from hair and makeup artistry to event design and decor styling.

Our goal is to simplify the process of discovering, booking, and managing premium beauty and event services. Whether preparing for a wedding, photoshoot, or an elegant soirée, LuxeHaven connects clients with professional stylists and designers who make every occasion unforgettable.

Built using the MERN Stack (MongoDB, Express.js, React, and Node.js), this application offers a seamless and responsive experience for both clients and service providers.

## Installation

To install this project:

1. Clone the repo

```sh
 git clone git@github.com:EddieJae24/luxehaven.git
```

2. Install NPM packages

```sh
npm install
```
3. Build the project
```sh
npm run build
```
4. Start the development server
```sh 
npm run develop
```

## Usage

Please follow these instructions to use the Book Search application in a local environment:

Note: You must have MongoDB installed on your computer to run this application. If you do not have MongoDB installed, you can download it [here](https://www.mongodb.com/try/download/community).

    1. Open a terminal and cd into the project's root directory
    2. Type npm run build to build the project
    3. Type npm run develop to start the server in development mode
    4. Click on the login/signup button in the top right corner of the page
    5. Create an account or login with an existing account
    6. Explore hair, makeup decor and book events
